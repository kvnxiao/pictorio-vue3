# pictorio-vue3

A draw and guess game for the modern era.

## Project setup

Requires Python 3.X to build, otherwise `node-gyp` will complain!

```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
